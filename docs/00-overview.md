# Anvil of Ideas - Project Overview

Anvil of Ideas is a React based web application that allows users to take their ideas for businesses, products, or services and turn them into fully fleshed out business, marketing, and product plans. The Anvil of Ideas uses AI to help users generate ideas, refine their ideas, and create comprehensive plans. 

## Core Modules

- **Anvil App**: The main application that allows users to create and manage their ideas.
- **n8n**: The workflow automation tool that handles the AI interactions and data processing.
- **Supabase**: The database that stores user data and ideas.
- **Ultimate Website**: A FastAPI app that generates a complete website for the user's idea.
- **Ultimate Pitch Deck**: A FastAPI app that generates a complete pitch deck for the user's idea.
- **Vibe Coder**: (Coming soon) A module that takes the user's generated specs and creates a complete, working, proof of concept application for the user's idea.

## Tech Stack

- **Frontend**: React, TypeScript, Tailwind CSS
- **Backend**: Express, FastAPI, Python
- **Database**: Supabase, Drizzle ORM
- **Workflow Automation**: n8n
- **AI**: OpenAI, Ollama, Mistral, Qwen
